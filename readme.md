# 项目：可自动生成接口文档的工具
运行JDK要求：JDK1.8以上  
要求：
1. 能使用指定的路径，扫描下面的所有带@Controller的注解  
2. 根据里面的javadoc注解，生成md格式的接口文档





# 后期考虑追加功能
1. 可以生成pdf，word文档格式的接口文档。
2. 可以将一些配置外部化。
3. 做成有web界面的接口应用
4. web应用可以一键生成接口的请求模板，调用方直接在页面上填充必要的参数就可以发起请求了
5. 从数据库生成实体类，带注释，带校验框架，
6. 界面化通过豆生成DTO对象
7. 接口文档支持模板生成接口文档
> 可通过特殊字符串，标识对应区域，填充该区域的数据即可
